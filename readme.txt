== Description ==


The KumulAid plugin adds the possibility for charities to add a donate button on their website using a simple shortcode.

To add this simply follow the following steps:
1. Install the KumulAid plugin
2. Add the following shortcode to any page: `[kumulaid id="<ID_OR_SLUG>"]`

An example is: `[kumulaid id="congodorpen"]` or `[kumulaid id="98"]`

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/kumulaid` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Now you can use the shortcode as described in the description section

== Frequently Asked Questions ==

= Where do I find my ID? =

You get your ID when signing up at https://kumulaid.com
